## Lesson Structure Rules

### Precheck (3 questions, type: "precheck")
- Tests knowledge from previous lessons or prerequisites
- Designed to be completed in the first 10 minutes of class
- Each question has 4 options: 1 correct + 3 distractors with misconception tags
- Use standard misconception tags (see below)

### Teach (1-3 sections, type: "teach")
- Clear instructional text with at least one fully worked example
- Break complex ideas into numbered steps
- Use concrete numbers, not abstract variables in examples
- Include a "common mistake" callout where applicable

### Practice (5-7 questions, type: "practice")
- Each question has an exact correct answer (for deterministic scoring)
- Each question has 4 options: 1 correct + 3 distractors
- Every wrong option MUST have a `reason` tag describing the misconception
- Wrong options should be plausible — the kind of mistake a real student would make

## Standard Misconception Tags for Algebra

Use these tags for the `reason` field on wrong options:
- `added-instead-of-subtracted` — performed addition when subtraction was needed (or vice versa)
- `multiplied-instead-of-divided` — used multiplication when division was needed
- `sign-error` — wrong sign on the answer (positive vs negative)
- `order-of-operations` — applied operations in wrong order
- `distributive-error` — failed to distribute correctly (e.g., 2(x+3) = 2x+3)
- `combining-unlike-terms` — added terms that can't be combined (e.g., 2x + 3 = 5x)
- `inverse-error` — used wrong inverse operation
- `fraction-error` — mishandled fractions or reciprocals
- `equation-setup-error` — set up the equation incorrectly
- `graphing-axis-swap` — swapped x and y
- `slope-calculation-error` — ran instead of rise, or sign error in slope
- `substitution-error` — substituted incorrectly
- `elimination-error` — added equations when should subtract
- `arithmetic-error` — simple calculation mistake
- `random-guess` — no clear pattern

## Pacing

Each lesson should be completable in a ~50-minute class period:
- Precheck: 10 minutes (3 questions)
- Teach: 15-20 minutes (instruction + worked examples)
- Practice: 20-25 minutes (5-7 questions with feedback)

## Week 1 Progression (Algebra 1 — Linear Equations)

A typical Week 1 algebra progression:
- Day 1: One-step addition/subtraction equations (x + a = b, x - a = b)
- Day 2: One-step multiplication/division equations (ax = b, x/a = b)
- Day 3: Two-step equations (ax + b = c)
- Day 4: Two-step equations with negative coefficients
- Day 5: Mixed practice + review of week's concepts
