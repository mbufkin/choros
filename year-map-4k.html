Documentation as Single Source

Model never goes outside uploaded docs. Time drifts — content does not. If a teacher uploads only "Algebra," the system produces output grounded in "Algebra" and nothing more. Full transparency: the system shows exactly what information it has and what is missing.

Model Tiers

  • Tier 1 (8–14B): Scoring + gap reports
  • Tier 2 (32–35B): Lessons, quizzes, pacing, remediation
  • Tier 3 (70B+): Crystallization, MOY, clustering
  • Build for Tier 2 first — prove it on 32B local hardware.

Student vs. Non-Student Split

  • Frontier model: Crystallization, MOY generation, pre-assessments — batch jobs, non-student-facing
  • Local model (Lenovo): Lesson delivery, quiz scoring, misconception tagging — student-facing
  • Deterministic: Mastery scoring — never AI

Audit Log — Non-Negotiable

  • Every teacher action modifying state is recorded
  • Crystallization approved? Logged.
  • Lesson bundle accepted? Logged.
  • Pacing adjusted? Logged with reason.
  • Protects the teacher and the system.
